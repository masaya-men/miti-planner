# FAB言語セレクター（円弧展開）設計書

## 概要

モバイルFABメニュー内の言語切替を、サイクル式（ja→en→zh→ko）から **円弧展開式セレクター** に改善する。Globeボタンを中心に左側へ扇形に4言語チップがspring staggeredアニメーションで飛び出す。

## 動機

- 現在のサイクル式は「今何語か」「次は何語か」が分かりにくい
- ラベル文言の出し入れによるちらつきを回避したい
- ネイティブアプリ風の楽しいモーションでUX向上

## 設計

### トリガーと構造

- FABメニュー内の **Globeボタン（🌐）をタップ** → 言語セレクターが展開/閉じる
- 「言語」ラベルとGlobeボタンは **位置・スタイルともに変化なし**（そのまま維持）
- Globeボタンの中心を起点に、**左側に円弧状に4つの言語チップが出現**
- FABメニューの他項目は表示を維持（閉じない）

### 円弧レイアウト

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| 半径 | 60px | Globeボタン中心→チップ中心 |
| 角度範囲 | 150°〜210° | 左方向、上下対称の扇形 |
| チップサイズ | 36×36px | 円形（border-radius: 50%） |
| チップ配置 | 上から: 한(150°), 中(170°), EN(190°), 日(210°) | 均等4分割 |

### チップスタイル

- **選択中の言語**: `bg-app-blue text-white` + 微妙なshadow
- **非選択**: `bg-black/70 text-white/90 backdrop-blur-sm`
- **テキスト**: 各言語の短縮表記（日 / EN / 中 / 한）、font-weight: 600、font-size: 13px

### アニメーション

#### 展開時（spring staggered fan-out）

1. **Globeパルス**: scale 1→1.15→1（120ms）— 展開の予告
2. **チップ飛び出し**: 中心（scale: 0, opacity: 0）→最終位置（scale: 1, opacity: 1）
   - spring設定: `stiffness: 380, damping: 15` — 到着時に1〜2回バウンド
   - stagger: 50ms間隔で順番に出現
3. **回転**: 各チップ出現時にrotate -15°→0°の軽い回転

#### 選択時

1. タップしたチップ: **scale 1→1.3**（100ms）→ フェードアウトしつつ中心へ
2. 他のチップ: **逆staggerで中心に吸い込まれる**（最後→最初の順）
3. その後: 既存のトランジションオーバーレイ（鉛筆アニメーション）が実行

#### 閉じる時（言語未選択でFABメニューを閉じた場合）

- 逆stagger（最後に出たものから順に中心へ戻る）
- spring snappy（既存の`SPRING.snappy`）で素早く収束

### motionTokens追加

```typescript
// SPRING に追加
bouncy: { type: 'spring', stiffness: 380, damping: 15 }
```

### 変更対象ファイル

| ファイル | 変更内容 |
|---------|---------|
| `src/components/MobileFAB.tsx` | 言語行のonClick変更、円弧チップコンポーネント追加 |
| `src/tokens/motionTokens.ts` | `SPRING.bouncy` 追加 |

### 変更しないもの

- FABメニュー全体の構造・他のメニュー項目
- 言語切替のロジック（i18n.changeLanguage + setContentLanguage）
- トランジションオーバーレイ（既存の鉛筆アニメーション）
- 「言語」ラベルの表示/非表示（常に表示のまま）

## 位置計算

```
// Globeボタン中心を原点(0,0)として
x = radius * cos(angle)
y = radius * sin(angle)

// 150°〜210°は左方向（cos < 0）
// 角度は数学座標系（反時計回り）
```

## エッジケース

- **画面回転**: fixed positionのFABなので問題なし
- **チップ同士の重なり**: 半径60px + 36pxチップで20°間隔 → 十分な間隔
- **連打防止**: 展開中の再タップは無視（AnimatePresence mode="wait"）

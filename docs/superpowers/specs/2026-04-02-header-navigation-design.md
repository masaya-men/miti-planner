# 表ヘッダーナビゲーション機能 — 設計書

## 概要

タイムライン表のヘッダー（Phase / Time / Mechanic列）をクリック可能にし、長いタイムラインでの素早い移動を実現する。フェーズへのジャンプ、時刻へのジャンプ、攻撃名での検索ジャンプの3機能を提供する。

## ユースケース

1. **長いタイムラインで迷子になったとき** — 「P3の途中に戻りたい」「さっき見てた時間に戻りたい」
2. **特定の攻撃を探して軽減を配置するとき** — 「ディヴァインダイブの軽減を確認したい」

## 機能詳細

### 1. フェーズ列ヘッダー（PC限定）

- **トリガー**: ヘッダーの「Phase」部分をクリック
- **動作**: フェーズ選択ドロップダウンを表示
  - コンテンツに含まれるフェーズ名を一覧表示
  - フェーズを選択するとフェーズ開始位置（`startTime`）にスムーズスクロール
  - ESCキーで閉じる
- **列折りたたみ機能**:
  - ドロップダウン内に折りたたみトグルを配置
  - 折りたたみ時: フェーズ列を非表示にし、数pxの細いハンドルを残す
  - ハンドルクリックで再展開
  - 用途: フェーズの概念がないコンテンツで表を広く使う

### 2. 時間列ヘッダー（PC限定）

- **トリガー**: ヘッダーの「Time」部分をクリック
- **動作**: 時間入力フィールドを表示
  - オートフォーカス（開いた瞬間に入力可能）
  - 入力形式:
    - 秒数: `120` → 2:00にジャンプ
    - 分:秒: `2:00` → 2:00にジャンプ
    - コロン有無で自動判定
  - Enterで該当時刻にスムーズスクロール
  - ESCキーで閉じる

### 3. 敵の攻撃列ヘッダー（PC + モバイル対応）

- **トリガー**: ヘッダーの「Mechanic」部分をクリック
- **動作**: 検索付き2段階ドロップダウンを表示

#### 1段目: 攻撃名リスト
- AA（オートアタック）を除外した一意な攻撃名リスト
- 各攻撃名の右に出現回数を表示（例: `×3 ▸`）
- 検索ボックスにオートフォーカス
- テキスト入力で攻撃名を絞り込み

#### 2段目: 出現箇所サブリスト
- 1段目で複数回出現する攻撃を選択すると、右にサブリストを展開
- 各出現箇所を「フェーズ名 + 時刻」で表示（例: `Phase 2  04:28`）
- 出現箇所を選択するとその位置にスムーズスクロール
- 出現が1回のみの攻撃は即ジャンプ（サブリスト不要）

#### モバイル対応
- モバイルでもMechanic列ヘッダーは操作可能
- ドロップダウンは画面幅に合わせたレイアウト調整

## 共通仕様

### インジケータ
- Phase / Time / Mechanic の全ヘッダーに統一された▼アイコンを表示
- クリック可能であることを視覚的に示す

### テーマ対応
- ライト/ダークテーマ両対応
- 既存テーマ変数（`app-text`, `app-bg`, `app-surface2`, `app-border`等）を使用

### キーボード操作
- ESCキー: 全ドロップダウン共通で閉じる
- Enter: 時間入力フィールドで確定

### スクロール
- 既存の `scrollContainerRef.scrollTo({ top, behavior: 'smooth' })` を利用
- `timeToYMap` で時刻からY座標を取得
- コンパクトモード（`hideEmptyRows`）でも正しく動作

### i18n
- 全UIテキストは翻訳キー経由（ハードコーディング禁止）

### レスポンシブ
- Phase列ヘッダー: PC（md以上）のみ
- Time列ヘッダー: PC（md以上）のみ
- Mechanic列ヘッダー: PC + モバイル対応

## 技術的な実装方針

### 既存コードとの統合
- ヘッダーは `Timeline.tsx` 内の `headerRef` 配下（行1618付近）
- スクロールは `scrollContainerRef` と `timeToYMap` を活用
- フェーズデータは Zustand store の `phases: Phase[]` から取得
- イベントデータは `eventsByTime` から攻撃名リストを生成

### コンポーネント構成
- `HeaderPhaseDropdown` — フェーズ選択 + 折りたたみトグル
- `HeaderTimeInput` — 時間入力ジャンプ
- `HeaderMechanicSearch` — 検索付き2段階ドロップダウン
- 各コンポーネントは独立した状態管理（開閉状態、選択状態）

### フェーズ列折りたたみの実装
- 折りたたみ状態を `useState` またはローカルストレージで管理
- 折りたたみ時: フェーズ列幅を0に、代わりに数pxのハンドル要素を表示
- ヘッダー・本体行・フェーズバーすべて連動して非表示に

# LP 3Dポータル リデザイン設計書

## 概要

現在の縦スクロール型ランディングページを、Three.jsによる全画面3Dポータルに全面刷新する。1画面で2つのアプリ（軽減プランナー / ハウジングツアー）への入口を3D空間内に配置し、没入感のある体験を提供する。

## コンセプト

**「3D空間に浮かぶポータル」** — サイバー × ミニマル幾何学 × ハウジングのオシャレ感を融合した世界観。暗い虚空にパーティクルと幾何学オブジェクトが漂い、2つのポータルがそれぞれの個性（シアン / アンバー）で光る。

## ページ構成

### メイン画面（100vh）
全画面Three.jsキャンバス。3D空間内に以下が存在する:

- **ロゴ「LoPo」**: 3Dテキストとして中央やや上方に浮遊。白〜淡グレー、両側からシアン/アンバーの光が当たる。ゆっくり上下にフロート
- **軽減プランナー ポータル（シアン）**: ロゴの一方に、奥行き・角度をつけて配置
- **ハウジングツアー ポータル（アンバー）**: ロゴの反対側に、奥行き・角度をつけて配置
- **言語切替**: 右上にオーバーレイ（HTMLレイヤー）
- **Preloader**: 初回アクセス時、3D読み込み中に表示

### ダイナミックフッター（スクロールで出現）
3Dポータル画面の下に配置。スクロールで3D空間から抜けてフッターが現れる。

## 3Dシーン詳細

### パーティクル
- 微細な光の粒が空間全体を漂う
- 基本色は白（空間の奥行きを表現）
- シアンの流れとアンバーの流れが各ポータルに向かって緩やかに流れる
- ポータル付近ではパーティクル密度が上がり、吸い寄せられる動き
- カスタムシェーダーで発光 + サイズ減衰（遠くの粒は小さく暗い）

### 幾何学オブジェクト
- **シアン側**: 正二十面体、八面体などの鋭角的なワイヤーフレーム。ゆっくり回転
- **アンバー側**: トーラス、球体などの丸みのある形状。ゆっくり回転
- フレネル効果で角度によって透明度が変化（ガラス/クリスタル質感）
- 各ポータル周囲に2〜3個が浮遊

### カメラ
- ゆっくり周回（2つのポータルとロゴを中心に）
- 緩やかに前後移動（奥行き感）
- マウス位置で視点が微動（PC/タブレットのみ）

### ポストプロセッシング
- **Bloom**: 発光体からの光の滲み。サイバー感の核
- **Depth of Field**: 手前と奥のボケで空間の奥行き表現
- **Chromatic Aberration**: 色収差で映像的な高級感
- **Vignette**: 画面端を暗くし視線を中央に集中

### マテリアル
- ガラス/クリスタル風の半透明シェーダー
- フレネル効果（角度で透明度変化）
- パーティクル: カスタムシェーダー（発光 + サイズ減衰）

## ポータルボタン（3D空間内UI）

### 軽減プランナー（シアン）
- 正二十面体のワイヤーフレームがゆっくり回転する中心にテキスト配置
- テキスト: 「軽減プランナー」（i18n対応、各言語で表示）
- シアンのパーティクルが周囲を漂う
- **ホバー時**: Bloom強化、パーティクルが加速して吸い寄せられる、図形の回転が速くなる
- **クリック時**: ワープ演出（zoom + blur）→ `/miti` へ遷移

### ハウジングツアー（アンバー）
- トーラスノットがゆっくり回転する中心にテキスト配置
- テキスト: 「ハウジングツアー」（i18n対応）
- アンバーのパーティクルが周囲を漂う
- **ホバー時**: 同様のリアクション（アンバー色）
- **クリック時**: 「準備中」のメッセージをオーバーレイ表示

## Preloader

- ミニマルなカウンター（0→100%）
- 背景は3Dシーンと同じ暗色で統一
- 完了後: パーティクルが画面全体に広がるように3D空間が現れる演出
- 2回目以降（sessionStorage判定）: 省略 or 短縮版

## カスタムカーソル

- **デフォルト**: 小さなシアンのドット + 細いリング
- **ホバー時**: リングが拡大、色がホバー対象に合わせて変化（シアン or アンバー）
- **クリック時**: リングからリップル波動
- **タッチデバイス**: 完全無効

## 言語切替

- 右上に固定（3Dキャンバスの上にHTMLオーバーレイ）
- `JP | EN | ZH | KO` の横並び、テキストのみ
- アクティブ言語: 白テキスト、それ以外: 半透明
- 極力目立たず、でも見つけられるサイズ

## ダイナミックフッター

- 3Dポータル画面（100vh）の下にスクロールで出現
- 境界演出: パーティクルが散って消えるトランジション
- フッター自体はダーク/ライトテーマに対応
- 内容:
  - 著作権表記（SQUARE ENIX）
  - ファンツール免責事項
  - Xシェアボタン、URLコピーボタン
  - リンク: Discord / X公式 / Privacy / Terms / Commercial Disclosure

## テーマ対応

### ダークモード
| 要素 | 値 |
|------|-----|
| 背景色 | `#0F0F10`（純黒） |
| パーティクル | 鮮やかに発光 |
| Bloom強度 | 強め |
| 幾何学図形 | シャープなワイヤーフレーム |
| UIテキスト | 白 |

### ライトモード
| 要素 | 値 |
|------|-----|
| 背景色 | `#1a1a2e`（ダークネイビー） |
| パーティクル | やや柔らかい光 |
| Bloom強度 | 控えめ |
| 幾何学図形 | やや透明度上げ、ソフト |
| UIテキスト | 白（空間が暗いため） |
| フッター | ライト背景に切替 |

## デザイントークン（CSS変数）

```css
/* LP専用アクセントカラー */
--color-portal-cyan: #00D4FF;
--color-portal-cyan-glow: #00D4FF80;
--color-portal-cyan-dim: #00D4FF1A;
--color-portal-amber: #FFB347;
--color-portal-amber-glow: #FFB34780;
--color-portal-amber-dim: #FFB3471A;

/* LP背景（テーマ別） */
--color-lp-bg-dark: #0F0F10;
--color-lp-bg-light: #1a1a2e;

/* LP UI */
--color-lp-text: #F0F0F0;
--color-lp-text-muted: #a1a1aa;
```

全てCSS変数で定義し、色味の調整を容易にする。Three.jsシェーダー内でもこれらの値を参照する仕組みを入れる。

## パフォーマンス戦略

| 環境 | パーティクル数 | ポストプロセス | 幾何学オブジェクト |
|------|-------------|-------------|--------------|
| PC | 500,000+ | フル（Bloom + DoF + CA + Vignette） | 高ポリゴン |
| タブレット | 200,000 | Bloom + Vignette | 中ポリゴン |
| スマホ | 80,000 | Bloom のみ | 低ポリゴン、数削減 |

- デバイス判定: `navigator.maxTouchPoints`、画面幅、GPU性能（WebGL拡張で推定）
- FPSモニタリング: 30fps以下が続く場合はパーティクル数を動的に削減
- テクスチャ: 最小限（パーティクルはシェーダーで生成）

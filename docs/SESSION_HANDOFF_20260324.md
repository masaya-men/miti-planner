## 今回のセッションでの重要な設計判断

1. **保存は2層構造** — localStorage(常時無音) + Firestore(間引き、ログインユーザーのみ、未実装)
2. **コンテンツ切替時にパーティ・軽減クリア** — 毎回パーティ編成から始める
3. **チュートリアル発火タイミング** — 初めて軽減表を自作しようとしたとき（共有リンクからの初訪問者には出さない）
4. **サイドバーのデフォルト** — プラン0件なら強制オープン、それ以外は前回の状態を記憶
5. **動画デモはMP4** — 静的ファイル配信なのでFirestore無料枠に影響なし
6. **UIは白黒のみ** — デザイン整えフェーズまでアクセントカラー禁止

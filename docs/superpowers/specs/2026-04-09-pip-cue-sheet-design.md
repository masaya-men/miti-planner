# PiP カンペビュー 設計書

## 概要

自分のジョブの軽減配置だけを時系列で一覧表示する「カンペ」ビュー。
PCではDocument Picture-in-Picture APIで別窓表示し、ゲーム画面の上に透過で重ねて使う。
スマホではフルスクリーンビューとして表示し、PS5の横にスマホを置いて使う。

## ユースケース

- PCゲーマー: FF14のゲーム画面上にカンペとして重ねて表示
- PS5ゲーマー: スマホを横に置いて自分の軽減タイミングを確認

## 表示内容

自分が軽減を配置している攻撃だけを時系列で表示する。
軽減を入れていない攻撃は表示しない（ユーザーはタイムラインを理解している前提）。

```
(3:25) レイ・オブ・ライト  [陣][フェイイルミ][セラフ][展開]
(4:10) テラスラッシュ      [陣]
```

- 左: 時間（mm:ss形式）
- 中央: 攻撃名（現在の言語設定に従う）
- 右: スキルアイコン列（その攻撃に配置した軽減スキル）
- ウィンドウ内でスクロール可能

### 攻撃名のインライン編集

- 攻撃名をダブルクリック/ダブルタップで編集モードに切り替え
- 自分用メモとして攻撃名を任意の文言に書き換え可能（例: 「散開」「頭割り」等）
- サーバーには保存しない
- localStorageにキー `pip-notes:{planId}` で保存（プラン単位で永続、ブラウザ内のみ）

## PC版: Document Picture-in-Picture

- `documentPictureInPicture.requestWindow()` で別窓を開く
- 非対応ブラウザ（Firefox, Safari）ではボタン非表示
- 初期サイズ指定あり（ユーザーはウィンドウ端ドラッグでリサイズ可能）

### 上部ツールバー（1行に収める）

```
[ ジョブアイコン ] [━━━●━━━ 透過率スライダー ] [ ✕ 閉じる ]
```

- **左: ジョブアイコン** — 現在表示中のメンバーのジョブアイコンが1つだけ小さく表示
  - クリックでPopover APIによるメニューが出現（スプリングアニメーション付き）
  - メニューにパーティメンバーのジョブアイコンをリスト表示
  - 選択するとそのメンバーの軽減に切り替え、メニュー自動クローズ
  - デフォルトは `myMemberId` のジョブ
- **中央: 透過率スライダー** — ウィンドウ背景の透過率を調整するrange input
- **右: 閉じるボタン** — PiPウィンドウを閉じる

## スマホ版: フルスクリーンカンペ

- 画面いっぱいのオーバーレイとして表示
- 透過率スライダーは不要（別端末で見る前提）
- ジョブ切替と閉じるボタンは上部に配置
- スクロール可能

## 起動ボタン

### PC版
- Timeline.tsxヘッダーの既存disabled `List` ボタン（行1714-1721）をPiPアイコンに差し替え
- `myMemberId` が未設定の場合はdisabled（自分のジョブ設定が前提条件）

### スマホ版
- モバイルのツールメニュー等に導線を追加

## 前提条件

- `myMemberId` が設定されていること（自分のジョブが未設定なら使えない）
- 該当メンバーに軽減が1つ以上配置されていること

## データフロー

```
useMitigationStore
  ├─ timelineMitigations (ownerId === 選択メンバーID でフィルタ)
  ├─ timelineEvents (軽減が配置されているイベントだけ抽出)
  ├─ partyMembers (ジョブ切替用リスト)
  └─ myMemberId (デフォルト選択)

localStorage
  └─ pip-notes:{planId} → { [eventId]: "ユーザーメモ" }
```

## 技術選定

| 技術 | 用途 |
|------|------|
| Document Picture-in-Picture API | PC別窓表示 |
| Popover API | ジョブ切替メニュー |
| CSS linear() スプリング | メニュー出現アニメーション |
| localStorage | 攻撃名メモの永続化 |

## 新規ファイル

| ファイル | 役割 |
|----------|------|
| `src/components/PipView.tsx` | カンペ表示のメインコンポーネント（PC/スマホ共通の表示ロジック） |
| `src/hooks/usePipNotes.ts` | localStorage メモ管理カスタムフック |

## 変更ファイル

| ファイル | 変更内容 |
|----------|----------|
| `Timeline.tsx` | disabled Listボタン → PiP起動ボタンに差し替え、PiPウィンドウ起動ロジック |
| モバイル導線 | ツールメニュー等にカンペビュー起動ボタン追加 |

## スコープ外

- 横スクロールタイムライン（音ゲーモード）— 将来別プロジェクト
- フルパーティ同時表示 — ジョブ切替で1人ずつ閲覧
- サーバーへのメモ保存 — localStorageのみ

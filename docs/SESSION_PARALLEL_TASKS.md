## タスク一覧

### 1. スマホ版: 軽減表ページにLoPoロゴが表示されていない
- **現象**: 人気ページにはLoPoロゴがあるが、軽減表（メインアプリ）ページにはスマホ時に表示されない
- **確認方法**: ブラウザのDevToolsでスマホ表示にして軽減表ページを開く
- **関連ファイル**: `src/components/Layout.tsx`（ヘッダー部分）、`src/components/LoPoButton.tsx`

### 2. スマホ版: 人気ページのLoPoロゴが他要素とぶつかって潰れている
- **現象**: 人気ページのスマホ表示でLoPoロゴが他の要素と重なってレイアウトが崩れている
- **関連ファイル**: `src/components/PopularPage.tsx` のヘッダー部分（※ガラスエフェクトのCSS は変更しない）
- **注意**: PopularPage.tsx のレイアウト修正のみ。ガラスエフェクトやスタイルクラスの変更は禁止

### 3. LoPoロゴ文字サイズ拡大（軽減表ヘッダー）
- **やりたいこと**: カプセル（LoPoButton）の大きさは変えずに、中の文字だけ大きくする
- **目安**: ランキング（人気）ページのLoPoロゴくらいの存在感にしたい
- **関連ファイル**: `src/components/LoPoButton.tsx` の `size="sm"` 設定周辺
- **注意**: カプセルの外形サイズは変更しない

### 4. プラン名編集のツールチップをカスタムデザインに
- **現象**: プラン名編集時のツールチップがブラウザデフォルトの見た目のまま
- **やりたいこと**: アプリ全体のデザインに合ったカスタムツールチップに統一
- **関連ファイル**: プラン名編集機能がある箇所を検索（`title=` や `tooltip` で grep）

### 5. PWA対応状況の確認（調査のみ）
- **やりたいこと**: スマホでホーム画面に追加してアプリとして使えるか確認
- **確認ポイント**:
  - `manifest.json` / `manifest.webmanifest` が存在するか
  - アイコン画像が適切に設定されているか
  - Service Worker が登録されているか
  - `<meta name="theme-color">` 等のPWA関連タグがあるか
- **アウトプット**: 調査結果と、不足しているものがあれば TODO.md に追記

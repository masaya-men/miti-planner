# イベントのドラッグ＆ドロップ移動（将来実装候補）

## 概要
タイムライン上のイベント（敵の攻撃行）を、ドラッグ＆ドロップで別の時刻に移動できる機能。

## ユースケース
- テンプレートが存在しないコンテンツでユーザーが生成した後の微調整
- テンプレートの並び順が気に食わないときの整頓
- 例：リーサルスカージのTB行とAoE行を好みの順序に並べ替え

## 実装時の考慮事項
- ドロップ先の時刻計算（行のY座標→秒数への変換）
- 同じ時刻に既にイベントがある場合の挿入位置
- ダメージ値は移動しても変わらない（time のみ更新）
- 軽減の効果範囲が変わるため、移動後にダメージ再計算が必要
- Undo/Redo 対応（pushHistory）
- モバイル対応（タッチD&D）
- 軽減のD&Dは @dnd-kit で実装済み。同じライブラリで統一可能

## 現状の代替手段
- イベントのコピペ機能で対応可能
- EventModal で time を直接編集することも可能

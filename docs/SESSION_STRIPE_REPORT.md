# Stripe対応セッション報告書（2026-03-27）

## 背景・経緯

Stripeから「ウェブサイトが基準を満たしていない」とのメールが届き、決済受け付け機能が停止された。
Ko-fi経由の支援金受け取りが不可の状態。

### Stripeが求めた対応
1. ウェブサイトがアクセス可能でログイン不要であること → **OK（lopoly.appは公開済み）**
2. 企業名・商品内容がStripe登録と一致すること → **対応済み（下記参照）**
3. 「特定商取引法に基づく表記」ページがウェブサイト内に存在すること → **今回作成**

### Stripe側で実施した設定変更
- **アカウント名**: Ko-fi → **LoPo** に変更済み
- **ウェブサイトURL**: ko-fi.com/lopoly → **lopoly.app** に変更済み
- **問い合わせ用Gmail**: **lopoly.contact@gmail.com** を新規作成済み

---

## 実装内容

### 新規作成: 特定商取引法に基づく表記ページ（`/commercial`）

既存のプライバシーポリシー・利用規約と同じ `LegalPageLayout` を使用したテーブル形式のページ。

#### 記載内容

| 項目 | 内容 |
|------|------|
| 販売事業者 | 請求があった場合は遅滞なく開示いたします |
| 所在地 | 請求があった場合は遅滞なく開示いたします |
| 電話番号 | 請求があった場合は遅滞なく開示いたします |
| メールアドレス | lopoly.contact@gmail.com |
| 運営責任者 | 請求があった場合は遅滞なく開示いたします |
| サービス内容 | FF14向けファンツール「LoPo」の運営。軽減プランナー等のWebツールを無料で提供 |
| 販売価格 | 任意の金額による自発的な支援（$3〜） |
| 追加手数料 | なし |
| 決済方法 | クレジットカード（Ko-fi を通じた決済） |
| 決済時期 | 支援時に即時決済 |
| サービス提供時期 | 本サービスは無料で即時利用可能です。支援は対価ではなく任意の寄付です |
| 返品・返金 | 支援金の性質上、返金には対応しておりません |

※個人事業主は2022年改正特商法により、氏名・住所・電話番号・運営責任者を「請求があった場合は遅滞なく開示」で省略可能。

## 未コミット・デプロイ前

このセッションではコミット・デプロイを行っていない。以下の手順で進めてください。

### デプロイ後にやること

1. **lopoly.app/commercial** にアクセスしてページが正しく表示されることを確認
2. **Stripeダッシュボード** → 設定 → ビジネス → ビジネスの詳細 → 「ウェブサイトを提出する」から再提出
3. 2営業日以内に審査結果がメールで届く
